# AlphaZero for Hungry Geese

## Requirements
* Python >= 3.7
* PyTorch >= 1.7.0
* coloredlogs
* tqdm
* kaggle_environments
* requests

## Quickstart
### Train
```
python main.py
```
### Submit to Kaggle
Run submit.ipynb

## Features
* Data augmentation by rotating and flipping the observations
* Multiprocessing for self-play
* Multi-GPU use for self-play and training
